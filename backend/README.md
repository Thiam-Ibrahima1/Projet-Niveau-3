# Exercice 2.3 : API Feveo avec MongoDB Atlas

API Node.js/Express complète avec authentification JWT, gestion des tâches et MongoDB Atlas.

## Caractéristiques

- Authentification avec JWT
- Enregistrement et connexion d'utilisateurs
- Gestion complète des tâches (CRUD)
- MongoDB Atlas pour la persistance des données
- Relation user → tasks (un utilisateur a plusieurs tâches)
- Validation des schémas Mongoose
- Gestion d'erreurs robuste
- Tests automatisés avec Jest

## Installation

### Prérequis

- Node.js v16+
- npm
- Compte MongoDB Atlas
- Postman
